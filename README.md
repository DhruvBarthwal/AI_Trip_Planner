# AI Trip Planner

An **AI-powered trip planning** system that generates personalized, day-wise travel itineraries based on destination, group type, and number of people.

Unlike simple chat-based planners, this system focuses on structured planning, constraint awareness, and clear, actionable itineraries that users can actually follow.

## Features

 Destination-based trip planning

 Supports different group types:

- Family

- Group

- Solo / Couple

 Day-wise itinerary generation

 Hotel recommendations aligned with trip duration

 Places to visit per day

 Suggested number of days for the trip

 Clean, readable, structured output (not raw paragraphs)

 ## How It Works

The planner takes a small set of user inputs and generates a complete itinerary by reasoning over multiple constraints.

### Inputs

- Destination (city / place)

- Number of people

- Group type (family, group, solo, couple)

### Outputs

- Total trip duration

- Day-by-day plan

- Places assigned per day

- Suggested hotels

- Balanced pacing (no overload)
